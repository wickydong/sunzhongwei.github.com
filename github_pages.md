# GitHub Pages


## Use markdown in GitHub Pages

直接创建后缀名字为 .md, 或者 .markdown 的文件即可。

但是首行需要加上

```markdown
---
title: This will be used as the title-tag of the page head
---
```

否则，Markdown 将不会被转换成  HTML.

所以，最好是 copy template.md 生存新文件。
